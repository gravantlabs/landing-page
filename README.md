# Gravant Labs — Landing Page

Static landing page for [gravantlabs.com](https://gravantlabs.com).

## Files

- `index.html` — Main page
- `privacy.html` — Privacy policy
- `terms.html` — Terms of service
- `css/` — Styles
- `js/` — Animations, consent banner, interactions

## Deploy

Static HTML. Drop into any host (Cloudflare Pages, Netlify, Vercel, or plain S3).
